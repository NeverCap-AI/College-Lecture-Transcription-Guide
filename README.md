### ğŸ”§ Quick Setup: Whisper (Free & Unlimited)
```bash
# 1. Install Python (if not already installed)
# Download from python.org

# 2. Install Whisper
pip install openai-whisper

# 3. Basic usage
whisper lecture01.mp3 --model medium

# 4. Advanced options
whisper lecture01.mp3 \
  --model medium \
  --language en \
  --output_format srt \
  --output_dir ./transcripts

# 5. With custom vocabulary (reduce errors)
whisper lecture01.mp3 \
  --initial_prompt "This lecture covers mitochondria, photosynthesis, and cellular respiration by Professor Martinez"
```

**Processing Time:** ~20-30% of audio length on modern laptops
**Accuracy:** 90-95% on clear audio

### 5-Minute Quick Start

**Step 1:** Check if recording is allowed
- [ ] Review university recording policy
- [ ] Send permission email to professor (use template)
- [ ] Wait for approval (usually 24-48 hours)

**Step 2:** Record your first lecture
- [ ] Test your device before class
- [ ] Position within 20 feet of professor
- [ ] Use highest quality settings your storage allows
- [ ] Start recording 1-2 minutes before class

- Follow tool-specific instructions
- Add custom vocabulary (professor names, technical terms)

- Save in organized folder structure
